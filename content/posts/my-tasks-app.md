O <kbd>MyTasks</kbd> é um aplicativo pessoal de gestão de tarefas. O projeto foca em uma separação clara de responsabilidades, garantindo segurança via autenticação JWT e uma interface mobile intuitiva e reativa. O projeto foi desenvolvido com o intuito de aprimorar meus conhecimentos em Backend e adquirir novas competências de Frontend e Mobile.

---

### Stack Tecnológica

- <kbd>Backend</kbd>: Desenvolvido com <abbr title="">Spring Boot</abbr>, utilizando <abbr title="">Spring Security</abbr> para proteção de dados e <abbr title="">Hibernate</abbr> para mapeamento objeto-relacional;

- <kbd>Mobile</kbd>: Interface moderna e performática construída com <abbr title="">Flutter</abbr>;

- <kbd>Banco de Dados</kbd>: Armazenamento relacional persistente com <abbr title="">MySQL</abbr>;

- <kbd>Documentação</kbd>: API documentada integralmente com <abbr title="">Swagger</abbr>;

---

### Diferenciais

- <kbd>Segurança</kbd>: Implementação de autenticação baseada em <abbr title="">Tokens JWT</abbr> e suporte a <abbr title="">Soft Deletes</abbr> para maior controle de dados;

- <kbd>Containerização</kbd>: Setup simplificado via <abbr title="">Docker</abbr> e <abbr title="">Docker Compose</abbr>, permitindo que o ambiente de desenvolvimento seja espelhado em produção sem conflitos;
